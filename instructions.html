Máquina de helado

Has de crear una máquina de helado, sigue los pados para hacer los que puedas en el tiempo que tenemos:

  1. En index.html, tienes un botón de encendido y 3 botones de sabores en la máquina. Cuando el usuario presiona el botón de encendido, el evento 'click' se lanzara para ese elemento. Hemos empezado a escribir un event listener. Encuentra la función anónima de callback para el event listener.
  2. Dentro del callback, enlaza el evento click para cada botón de sabor de la máquina.
  3. Después, pon isOn a true dentro del mismo callback.
  4. Cada botón de la máquina debe hacer algo. Usa las imagenes incluidas img/chocolate.jpg, img/vanilla.jpg, and img/swirl.jpg para crear un tag <img>, añade un atributo src, y adjuntalo al <div> de la cinta trasnsportadora del lado de la máquina.
  5. Abre index.html en el navegador y asegurate de que puedes poner la máquina en marcha con el botón de encendido. Asegúrate de que las imágenes se añaden a la cinta trasnportadora.
  6. Ahora, también queremos poder apagar la máquina. Añade la lógica para que los eventos se enlacen cuando la máquina se ponga en marcha y se desenlacen cuando la máquina se apague. Además asegúrate de que isOn cambia para reflejar de forma precisa el estado de la máquina.
  7. BONUS: Límpia el código de forma que con una sola función podamos crear cualquier sabor, dependiendo del botón presionado.

Asegúrate de usar las "browser developer tools" para debugar más facilmente mientras trabajas en este ejercicio. Comprueba errores usando console.log() para ver hasta donde llega el código, lo que hace y que valores tienen las variabñes durante la ejecución.
